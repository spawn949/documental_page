Introduction

This application parses the data output from the scientific program Seisan.

It has two main functions:

  1. Send the last 20 seismic events to the seismology website.
  2. Send emails to the people in the seismology list.

It takes this output and transform it in data wich can be understood from visitors giving the activity of the last 20 earthquakes.

Internally it uses the geospatial data given by Mr moreta and it has all the poligons wich wraps the political map from the Santo Domingo Island.

For calculate if the epicentre is in any poligon it uses the algorithm extrated from "https://sakseiw.wordpress.com"

The interface has the following components wich we will see them in next sections:

Mag Local

This is one of the subcomponents of the adio button component.

When this option is marked it will send to the ouput the local magnitud of the event.

Mag Coda

This is one of the subcomponents of the adio button component.

When this option is marked it will send to the ouput the coda magnitud of the event.

Mag MW

This is one of the subcomponents of the adio button component.

When this option is marked it will send to the ouput the mw magnitud of the event.

Mag Promedio

This is one of the subcomponents of the adio button component.

When this option is marked it will send to the ouput a promedio of the magnituds of the event.

Sentido

This one of the options in the check button component.

when checked it will send the infomation that this event was feeled by people close to the epicentre.

Enviar Correos a Sini

This check button option will send an email to the sini institution.

Enviar Correos a Todos

This option will send and email to all people in the email list.

Enviar Datos
